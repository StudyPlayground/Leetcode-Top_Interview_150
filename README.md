## 📌 진행방식

1. 각 회차의 시작 날에는 2-4문제를 선정합니다.
2. 선정된 문제를 풀고 코드를 업로드합니다.
3. 각 회차의 마지막 날에는 풀이 설명 시간을 갖습니다.

> - 파일명은 각 문제명의 폴더 안에 `닉네임`으로 작성합니다.
>   - 예: max.js
> - commit은 해당 문제의 폴더명으로 작성합니다.
>   - 예: Jump_Game_(Medium)
